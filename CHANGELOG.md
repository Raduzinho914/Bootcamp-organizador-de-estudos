# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato segue o padrão [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adota [Versionamento Semântico](https://semver.org/lang/pt-BR/).

---

## [1.0.0] - 2025-01-01

### Adicionado
- Dashboard com estatísticas de tarefas (total, pendentes, concluídas, atrasadas)
- CRUD completo de Matérias com cor personalizável
- CRUD completo de Tarefas com status, prioridade e prazo
- Marcação rápida de tarefa como concluída
- Filtro de tarefas por status, matéria e prioridade
- Detecção automática de tarefas atrasadas
- Grade semanal de sessões de estudo (CRUD)
- Cálculo automático de duração de cada sessão
- Interface web responsiva com tema escuro
- Testes automatizados com pytest-django
- Lint com ruff
- Pipeline de CI com GitHub Actions
